# Proyectos

## 1) Used Car Market Analysis – Streamlit App

Aplicación web interactiva para explorar datos de anuncios de vehículos usados mediante visualizaciones dinámicas.

### Objetivo
Analizar un dataset de anuncios de vehículos para identificar cómo variables como **precio, kilometraje, año y condición del vehículo** influyen en el valor de mercado.

### Análisis realizado
- Distribución de precios de vehículos
- Relación entre kilometraje y precio
- Comparación de precios según condición del vehículo
- Tendencias en publicaciones de anuncios

### Aplicación
La aplicación permite explorar el dataset mediante **gráficos interactivos y filtros dinámicos**.

## 2) Chicago Taxi Data Analysis

Análisis exploratorio de datos de viajes de taxi en Chicago para identificar patrones de demanda y evaluar el impacto del clima en la duración de los viajes.

### Objetivo
Analizar datos de viajes de taxi para:

- Identificar las empresas con mayor número de viajes
- Determinar los barrios con más finalizaciones de viajes
- Analizar el impacto del clima en la duración de viajes hacia el aeropuerto O'Hare

### Análisis realizado
- Consultas SQL para análisis de datos
- Exploración de datos con Python
- Visualización de resultados mediante gráficos

### Insights principales
- Algunas empresas concentran la mayor parte de los viajes
- Zonas céntricas presentan mayor volumen de finalizaciones
- Las condiciones climáticas pueden influir en la duración de los viajes
